Lab 1: Timing Leak

Create a program to crack the password of a server that leaks information via a timing side-channel. Your submission should be only a modified crackPassword.js file. Do not submit .zip files or anything else, or you will receive 0 (but you can re-submit).
The password server program is a simple server that checks a provided password letter by letter, and gives up when it sees a match. It has an extra sleep() in between each letter to make the timing differences extra obvious.
The point of this lab is to learn about side-channel attacks, and to get some practice with writing javascript code that uses async/await.
  • Download the files passwordServer.js and crackPassword.js.
  • Install the npm dependencies: 
    npm install express node-fetch
  • Start the password server: 
    node passwordServer.js
  • In a separate window, run the crack password program: 
    node crackPassword.js
    It will guess each letter of the alphabet as the password. See if any particular letter takes longer than others.
  • Edit the crackPassword.js and write code to figure out the password.
